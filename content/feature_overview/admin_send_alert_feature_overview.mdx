<a id="admin-send-alert-feature-overview"></a>


Sends a user generated alert to the monitoring system.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| message | string | Alert message body. <br/> <br/> The default value is ''. |
| label | string | Label to add to alert message. <br/> <br/> The default value is ''. |
| log_level | string | Alert message logging criteria. <br/> <br/> The supported values are: <br/> <br/> * fatal <br/> * error <br/> * warn <br/> * info <br/> * debug |
| options | map of string to strings | Optional parameters. <br/> <br/> The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| info | map of string to strings | Additional information. |
