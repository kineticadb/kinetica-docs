<a id="admin-ha-refresh-feature-overview"></a>


Restarts the HA processing on the given cluster as a mechanism of accepting
breaking HA conf changes. Additionally the cluster is put into read-only while
HA is restarting.


## Input Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| options | map of string to strings | Optional parameters. <br/> <br/> The default value is an empty map ( \{\} ). |


## Output Parameter Description


| Name | Type | Description |
| --- | --- | --- |
| info | map of string to strings | Additional information. |
