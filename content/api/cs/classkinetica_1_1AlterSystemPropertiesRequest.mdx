<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#acfc3b756114ef27bd448b894944ce4a0">Kinetica.alterSystemProperties</a>. </p>
<p>The <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#acfc3b756114ef27bd448b894944ce4a0">Kinetica.alterSystemProperties</a> endpoint is primarily used to simplify the testing of the system and is not expected to be used during normal execution. Commands are given through the <a className="el" href="#aa071e775ae1b4dd144350bf2447d19e0" title="Map containing the properties of the system to be updated.">AlterSystemPropertiesRequest.property\_updates\_map</a> whose keys are commands and values are strings representing integer values (for example '8000') or boolean values ('true' or 'false').</p>
