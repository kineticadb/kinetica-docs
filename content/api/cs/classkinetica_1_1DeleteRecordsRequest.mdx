<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#aa9a90dadf56669a4883a44c9553e5995">Kinetica.deleteRecords</a>. </p>
<p>Deletes record(s) matching the provided criteria from the given table. The record selection criteria can either be one or more <a className="el" href="#aecd0c0f7c0ee31fc00ac48c1294535f5" title="A list of the actual predicates, one for each select; format should follow the guidelines provided he...">DeleteRecordsRequest.expressions</a> (matching multiple records), a single record identified by <a className="el" href="/content/api/cs/structkinetica_1_1DeleteRecordsRequest_1_1Options#a4a820b0b7e7607a4017b9809f89a4b47">RECORD\_ID</a> options, or all records when using <a className="el" href="/content/api/cs/structkinetica_1_1DeleteRecordsRequest_1_1Options#a0fd3a52d6ed96cd6028b6dda177d9542">DELETE\_ALL\_RECORDS</a>. Note that the three selection criteria are mutually exclusive. This operation cannot be run on a view. The operation is synchronous meaning that a response will not be available until the request is completely processed and all the matching records are deleted.</p>
