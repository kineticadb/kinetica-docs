<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#ae8f4a10b254066875da694eab13e4b80">Kinetica.filterByBox</a>. </p>
<p>Calculates how many objects within the given table lie in a rectangular box. The operation is synchronous, meaning that a response will not be returned until all the objects are fully available. The response payload provides the count of the resulting set. A new resultant set which satisfies the input NAI restriction specification is also created when a <a className="el" href="#a1022e436a6a6faf224b5efe870edf07d" title="If provided, then this will be the name of the view containing the results, in [schema_name.">FilterByBoxRequest.view\_name</a> is passed in as part of the input payload.</p>
