<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#a2bc0ad61d786b878102a21366685f24e">Kinetica.filterByAreaGeometry</a>. </p>
<p>Calculates which geospatial geometry objects from a table intersect a named area of interest (NAI/polygon). The operation is synchronous, meaning that a response will not be returned until all the matching objects are fully available. The response payload provides the count of the resulting set. A new resultant set (view) which satisfies the input NAI restriction specification is created with the name <a className="el" href="#a15c9f1eb39559178830019e53022dd56" title="If provided, then this will be the name of the view containing the results, in [schema_name.">FilterByAreaGeometryRequest.view\_name</a> passed in as part of the input.</p>
