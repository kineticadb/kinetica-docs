<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#abfe6dfadda06fd3614bfff428f7ee142">Kinetica.alterTableColumns</a>. </p>
<p>Apply various modifications to columns in a table, view. The available modifications include the following:</p>
<p>Create or delete an <a href="/content/concepts/indexes/#column-index" target="_top">index</a> on a particular column. This can speed up certain operations when using expressions containing equality or relational operators on indexed columns. This only applies to tables.</p>
<p>Manage a table's columns&ndash;a column can be added, removed, or have its <a href="/content/concepts/types/" target="_top">type and properties</a> modified, including whether it is <a href="/content/concepts/dictionary_encoding/" target="_top">dictionary encoded</a> or not.</p>
