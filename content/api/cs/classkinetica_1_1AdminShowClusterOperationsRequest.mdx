<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#ad305a4e3f0d6361db445101822017aaf">Kinetica.adminShowClusterOperations</a>. </p>
<p>Requests the detailed status of the current operation (by default) or a prior cluster operation specified by <a className="el" href="#a25c9577e3118ae68162251be376f2fc4" title="Indicates which cluster operation to retrieve.">AdminShowClusterOperationsRequest.history\_index</a>. Returns details on the requested cluster operation.</p>
<p>The response will also indicate how many cluster operations are stored in the history.</p>
