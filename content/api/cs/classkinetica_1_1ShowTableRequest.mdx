<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#af70f60085dc4c30ab3ffd15ff4622837">Kinetica.showTable</a>. </p>
<p>Retrieves detailed information about a table, view, or schema, specified in <a className="el" href="#ab135222f57559b210d10c84647e68c0b" title="Name of the table for which to retrieve the information, in [schema_name.">ShowTableRequest.table\_name</a>. If the supplied <a className="el" href="#ab135222f57559b210d10c84647e68c0b" title="Name of the table for which to retrieve the information, in [schema_name.">ShowTableRequest.table\_name</a> is a schema the call can return information about either the schema itself or the tables and views it contains. If <a className="el" href="#ab135222f57559b210d10c84647e68c0b" title="Name of the table for which to retrieve the information, in [schema_name.">ShowTableRequest.table\_name</a> is empty, information about all schemas will be returned.</p>
<p>If the option <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a2ae4d0a37ed4436d0fbafd05708d050c">GET\_SIZES</a> is set to <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a26d1e9d5c77f4af051a6608a2d15fe95">TRUE</a>, then the number of records in each table is returned (in <a className="el" href="/content/api/cs/classkinetica_1_1ShowTableResponse#a6c46795af2df44cca0ccad279ad4ed1e">sizes</a> and <a className="el" href="/content/api/cs/classkinetica_1_1ShowTableResponse#af5087b3021e53a9b5471f9e9a965b226">full\_sizes</a>), along with the total number of objects across all requested tables (in <a className="el" href="/content/api/cs/classkinetica_1_1ShowTableResponse#a578985a9b0b015a974cc5d805c525413">total\_size</a> and <a className="el" href="/content/api/cs/classkinetica_1_1ShowTableResponse#aa883216b443525d0fb4bcffee5fa4b6f">total\_full\_size</a>).</p>
<p>For a schema, setting the <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a59111dd1d2e9f163ef342fa4809a610b">SHOW\_CHILDREN</a> option to <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a4c1fd82aae7425c6c47c4cab1a1c2c7f">FALSE</a> returns only information about the schema itself; setting <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a59111dd1d2e9f163ef342fa4809a610b">SHOW\_CHILDREN</a> to <a className="el" href="/content/api/cs/structkinetica_1_1ShowTableRequest_1_1Options#a26d1e9d5c77f4af051a6608a2d15fe95">TRUE</a> returns a list of tables and views contained in the schema, along with their corresponding detail.</p>
