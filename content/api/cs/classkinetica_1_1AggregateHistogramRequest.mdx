<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#ae228ebf36c03c4b2c5b5024461cefd8b">Kinetica.aggregateHistogram</a>. </p>
<p>Performs a histogram calculation given a table, a column, and an interval function. The <a className="el" href="#a3c5b0cd3915aa1ca98ea21f2fa1ffdb5" title="The size of each bin within the start and end parameters.">AggregateHistogramRequest.interval</a> is used to produce bins of that size and the result, computed over the records falling within each bin, is returned. For each bin, the start value is inclusive, but the end value is exclusive&ndash;except for the very last bin for which the end value is also inclusive. The value returned for each bin is the number of records in it, except when a column name is provided as a <a className="el" href="/content/api/cs/structkinetica_1_1AggregateHistogramRequest_1_1Options#a5dc75615230fc6a908d6617b5a949391">VALUE\_COLUMN</a>. In this latter case the sum of the values corresponding to the <a className="el" href="/content/api/cs/structkinetica_1_1AggregateHistogramRequest_1_1Options#a5dc75615230fc6a908d6617b5a949391">VALUE\_COLUMN</a> is used as the result instead. The total number of bins requested cannot exceed 10,000.</p>
<p>NOTE: The <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica" title="API to talk to Kinetica Database.">Kinetica</a> instance being accessed must be running a CUDA (GPU-based) build to service a request that specifies a <a className="el" href="/content/api/cs/structkinetica_1_1AggregateHistogramRequest_1_1Options#a5dc75615230fc6a908d6617b5a949391">VALUE\_COLUMN</a>. </p>
