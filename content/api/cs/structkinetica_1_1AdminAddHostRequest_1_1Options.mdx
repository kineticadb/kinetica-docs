<p>Supported units: K (thousand), KB (kilobytes), M (million), MB (megabytes), G (billion), GB (gigabytes); if no unit is provided, the value is assumed to be in bytes. Set <a className="el" href="#a240241d0db30d629d622974ab118d6c5">RAM\_LIMIT</a> to -1 to have no RAM limit.</p>
