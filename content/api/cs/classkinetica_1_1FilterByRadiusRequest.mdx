<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#a51db1dcb234610239f31f7689849e17c">Kinetica.filterByRadius</a>. </p>
<p>Calculates which objects from a table lie within a circle with the given radius and center point (i.e. circular NAI). The operation is synchronous, meaning that a response will not be returned until all the objects are fully available. The response payload provides the count of the resulting set. A new resultant set (view) which satisfies the input circular NAI restriction specification is also created if a <a className="el" href="#a1e4087e8a73c4f2b27a382f1c1eaa94d" title="If provided, then this will be the name of the view containing the results, in [schema_name.">FilterByRadiusRequest.view\_name</a> is passed in as part of the request.</p>
<p>For track data, all track points that lie within the circle plus one point on either side of the circle (if the track goes beyond the circle) will be included in the result.</p>
