<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#a700c5cf8bafb7b1213d19fe20613e829">Kinetica.filterByArea</a>. </p>
<p>Calculates which objects from a table are within a named area of interest (NAI/polygon). The operation is synchronous, meaning that a response will not be returned until all the matching objects are fully available. The response payload provides the count of the resulting set. A new resultant set (view) which satisfies the input NAI restriction specification is created with the name <a className="el" href="#acc2ff25d80773373aa9b0dd0eeb5eb1c" title="If provided, then this will be the name of the view containing the results, in [schema_name.">FilterByAreaRequest.view\_name</a> passed in as part of the input. </p>
