<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#a82c23966afc97e669b9b08abef7970d6">Kinetica.alterTier</a>. </p>
<p>Alters properties of an existing <a href="/content/rm/concepts/#storage-tiers" target="_top">tier</a> to facilitate <a href="/content/rm/concepts/" target="_top">resource management</a>.</p>
<p>To disable <a href="/content/rm/concepts/#watermark-based-eviction" target="_top">watermark-based eviction</a>, set both <a className="el" href="/content/api/cs/structkinetica_1_1AlterTierRequest_1_1Options#a1145841aef990d52edad62335b9d4600">HIGH\_WATERMARK</a> and <a className="el" href="/content/api/cs/structkinetica_1_1AlterTierRequest_1_1Options#a60574cb8690f70a3b4e3b19bb5807ac3">LOW\_WATERMARK</a> to 100.</p>
