<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of string constants for the parameter <a className="el" href="/content/api/cs/classkinetica_1_1AlterResourceGroupRequest#a2b339897ec44e8a234c42eb54d079a67" title="Optional map containing tier names and their respective attribute group limits.">AlterResourceGroupRequest.tier\_attributes</a>. </p>
<p>Optional map containing tier names and their respective attribute group limits. The only valid attribute limit that can be set is max\_memory (in bytes) for the VRAM and RAM tiers.</p>
<p>For instance, to set max VRAM capacity to 1GB per rank per GPU and max RAM capacity to 10GB per rank, use: {"{"}'VRAM':{"{"}'max\_memory':'1000000000'{"}"}, 'RAM':{"{"}'max\_memory':'10000000000'{"}"}{"}"}</p>
