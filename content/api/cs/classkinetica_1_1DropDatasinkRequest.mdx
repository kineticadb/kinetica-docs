<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cs/classkinetica_1_1Kinetica#a8e9edda06a969068cf03e6cb9979c3ac">Kinetica.dropDatasink</a>. </p>
<p>Drops an existing <a href="/content/concepts/data_sinks/" target="_top">data sink</a>.</p>
<p>By default, if any <a href="/content/concepts/table_monitors" target="_top">table monitors</a> use this sink as a destination, the request will be blocked unless option <a className="el" href="/content/api/cs/structkinetica_1_1DropDatasinkRequest_1_1Options#a066d81e4794fc3ef39e8b53be7d8b540">CLEAR\_TABLE\_MONITORS</a> is <a className="el" href="/content/api/cs/structkinetica_1_1DropDatasinkRequest_1_1Options#a76035b78f984f4934a201ba0cb5750cd">TRUE</a>.</p>
