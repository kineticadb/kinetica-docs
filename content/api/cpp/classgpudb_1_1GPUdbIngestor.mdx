<div className="memdoc">
<p>Ensures that all queued records are inserted into the database. </p>
<p>If an error occurs while inserting the records from any queue, the recoreds will no longer be in that queue nor in the database; catch GPUdbInsertException&lt;T&gt; to get the list of records that were being inserted if needed (for example, to retry). Other queues may also still contain unflushed records if this occurs. </p>
