<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cpp/classgpudb_1_1GPUdb#aa11e77cd54b2d748b261fccbb9508765">GPUdb::adminShowClusterOperations</a>. </p>
<p>Requests the detailed status of the current operation (by default) or a prior cluster operation specified by <a className="el" href="#ac6af84e10bd8b42efba940403e60583a">historyIndex</a>. Returns details on the requested cluster operation.</p>
