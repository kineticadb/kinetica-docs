<h2 className="groupheader">Detailed Description</h2>
<div className="textblock">
<p>A set of parameters for <a className="el" href="/content/api/cpp/classgpudb_1_1GPUdb#a827d2d54888b00d0dcc69599325de9cf">GPUdb::createTable</a>. </p>
<p>Creates a new table with the given type (definition of columns). The type is specified in <a className="el" href="#a0352e9c8a05a6b02fd1676ecfb5f10f3">typeId</a> as either a numerical type ID (as returned by <a className="el" href="/content/api/cpp/classgpudb_1_1GPUdb#abaf7eca0d3c131b8fede4850d51f1a1a">GPUdb::createType</a>) or as a list of columns, each specified as a list of the column name, data type, and any column attributes.</p>
<p>Example of a type definition with some parameters: </p>
<div className="fragment">
<div className="line">\[</div>
<div className="line"> \[<span className="stringliteral">&quot;id&quot;</span>, <span className="stringliteral">&quot;int8&quot;</span>, <span className="stringliteral">&quot;primary\_key&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;dept\_id&quot;</span>, <span className="stringliteral">&quot;int8&quot;</span>, <span className="stringliteral">&quot;primary\_key&quot;</span>, <span className="stringliteral">&quot;shard\_key&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;manager\_id&quot;</span>, <span className="stringliteral">&quot;int8&quot;</span>, <span className="stringliteral">&quot;nullable&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;first\_name&quot;</span>, <span className="stringliteral">&quot;char32&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;last\_name&quot;</span>, <span className="stringliteral">&quot;char64&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;salary&quot;</span>, <span className="stringliteral">&quot;decimal&quot;</span>\],</div>
<div className="line"> \[<span className="stringliteral">&quot;hire\_date&quot;</span>, <span className="stringliteral">&quot;date&quot;</span>\]</div>
<div className="line">\]</div>
</div>
<p> Each column definition consists of the column name (which should meet the standard <a href="/content/concepts/tables/#table-naming-criteria" target="_top">column naming criteria</a>), the column's <a href="/content/concepts/types/#types-chart" target="_top">specific type</a> (int, long, float, double, string, bytes, or any of the properties map values from <a className="el" href="/content/api/cpp/classgpudb_1_1GPUdb#abaf7eca0d3c131b8fede4850d51f1a1a">GPUdb::createType</a>), and any <a href="/content/concepts/types/#types-data-handling" target="_top">data handling</a>, <a href="/content/concepts/types/#types-data-keys" target="_top">data key</a>, or <a href="/content/concepts/types/#types-data-replace" target="_top">data replacement</a> properties.</p>
<p>A table may optionally be designated to use a <a href="/content/concepts/tables/#replication" target="_top">replicated</a> distribution scheme, or be assigned: <a href="/content/concepts/tables/#foreign-keys" target="_top">foreign keys</a> to other tables, a <a href="/content/concepts/tables/#partitioning" target="_top">partitioning</a> scheme, and/or a <a href="/content/rm/concepts/#tier-strategies" target="_top">tier strategy</a>. </p>
