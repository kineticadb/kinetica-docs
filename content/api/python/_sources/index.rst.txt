Kinetica Python API
===================

This project contains the source code of the Kinetica Python API. For an
overview of the Kinetica architecture, please visit
`https://docs.kinetica.com/ <../../../>`__.

For a tutorial, see the `Python Developer Guide <../../../guides/python_guide/>`__.
