<div className="block">A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#getRecordsBySeries(com.gpudb.protocol.GetRecordsBySeriesRequest)"><code>GPUdb.getRecordsBySeries</code></a>. <p> Retrieves the complete series/track records from the given <a href="#getWorldTableName()"><code>worldTableName</code></a> based on the partial track information contained in the <a href="#getTableName()"><code>tableName</code></a>. </p><p> This operation supports paging through the data via the <a href="#getOffset()"><code>offset</code></a> and <a href="#getLimit()"><code>limit</code></a> parameters. </p><p> In contrast to <a href="/content/api/java/com/gpudb/GPUdb#getRecords(com.gpudb.protocol.GetRecordsRequest)"><code>GPUdb.getRecords</code></a> this returns records grouped by series/track. So if <a href="#getOffset()"><code>offset</code></a> is 0 and <a href="#getLimit()"><code>limit</code></a> is 5 this operation would return the first 5 series/tracks in <a href="#getTableName()"><code>tableName</code></a>. Each series/track will be returned sorted by their TIMESTAMP column.</p></div>
