<div className="section detail" id="MODE">
<h3>MODE</h3>
<div className="member-signature"><span className="modifiers">public static final</span>&nbsp;<span className="return-type"><a href="https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html" title="class or interface in java.lang" className="external-link">String</a></span>&nbsp;<span className="element-name">MODE</span></div>
<div className="block">
The mode describes what rows of the tables being unioned will be retained. Supported values: 
<ul>
<li><a href="#UNION_ALL"><code>UNION\_ALL</code></a>: Retains all rows from the specified tables. </li>
<li><a href="#UNION"><code>UNION</code></a>: Retains all unique rows from the specified tables (synonym for <a href="#UNION_DISTINCT"><code>UNION\_DISTINCT</code></a>). </li>
<li><a href="#UNION_DISTINCT"><code>UNION\_DISTINCT</code></a>: Retains all unique rows from the specified tables. </li>
<li><a href="#EXCEPT"><code>EXCEPT</code></a>: Retains all unique rows from the first table that do not appear in the second table (only works on 2 tables). </li>
<li><a href="#EXCEPT_ALL"><code>EXCEPT\_ALL</code></a>: Retains all rows(including duplicates) from the first table that do not appear in the second table (only works on 2 tables). </li>
<li><a href="#INTERSECT"><code>INTERSECT</code></a>: Retains all unique rows that appear in both of the specified tables (only works on 2 tables). </li>
<li><a href="#INTERSECT_ALL"><code>INTERSECT\_ALL</code></a>: Retains all rows(including duplicates) that appear in both of the specified tables (only works on 2 tables). </li>
</ul>
 The default value is <a href="#UNION_ALL"><code>UNION\_ALL</code></a>.
</div>
