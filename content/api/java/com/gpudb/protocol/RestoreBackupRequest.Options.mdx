<div className="member-signature"><span className="modifiers">public static final</span>&nbsp;<span className="return-type"><a href="https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html" title="class or interface in java.lang" className="external-link">String</a></span>&nbsp;<span className="element-name">REINGEST</span></div>
<div className="block">
Behavior to apply when restoring table data. Supported values: 
<ul>
<li><a href="#TRUE"><code>TRUE</code></a>: Restore table data by re-ingesting it. This is the default behavior if the cluster topology differs from that of the contained backup. </li>
<li><a href="#FALSE"><code>FALSE</code></a>: Restore the persisted data files directly. </li>
</ul>
 The default value is <a href="#FALSE"><code>FALSE</code></a>.

<div className="section detail" id="RESTORE_POLICY">
<h3>RESTORE\_POLICY</h3>
<div className="member-signature"><span className="modifiers">public static final</span>&nbsp;<span className="return-type"><a href="https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/lang/String.html" title="class or interface in java.lang" className="external-link">String</a></span>&nbsp;<span className="element-name">RESTORE\_POLICY</span></div>
<div className="block">
Behavior to apply when any database object to restore already exists. Supported values: 
<ul>
<li><a href="#NONE"><code>NONE</code></a>: If an object to be restored already exists with the same name, abort and return error. </li>
<li><a href="#REPLACE"><code>REPLACE</code></a>: If an object to be restored already exists with the same name, replace it with the backup version. </li>
<li><a href="#RENAME"><code>RENAME</code></a>: If an object to be restored already exists with the same name, move that existing one to the schema specified by <a href="#RENAMED_OBJECTS_SCHEMA"><code>RENAMED\_OBJECTS\_SCHEMA</code></a>. This policy does not apply to non-schema objects. </li>
</ul>
 The default value is <a href="#NONE"><code>NONE</code></a>.
</div>
