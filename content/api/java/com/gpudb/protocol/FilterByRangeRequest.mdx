<div className="block">
A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#filterByRange(com.gpudb.protocol.FilterByRangeRequest)"><code>GPUdb.filterByRange</code></a>. 
<p> Calculates which objects from a table have a column that is within the given bounds. An object from the table identified by <a href="#getTableName()"><code>tableName</code></a> is added to the view <a href="#getViewName()"><code>viewName</code></a> if its column is within \[<a href="#getLowerBound()"><code>lowerBound</code></a>, <a href="#getUpperBound()"><code>upperBound</code></a>\] (inclusive). The operation is synchronous. The response provides a count of the number of objects which passed the bound filter. Although this functionality can also be accomplished with the standard filter function, it is more efficient. </p>
<p> For track objects, the count reflects how many points fall within the given bounds (which may not include all the track points of any given track).</p>
