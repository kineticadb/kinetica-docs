A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#filterByArea(com.gpudb.protocol.FilterByAreaRequest)"><code>GPUdb.filterByArea</code></a>. 
<p> Calculates which objects from a table are within a named area of interest (NAI/polygon). The operation is synchronous, meaning that a response will not be returned until all the matching objects are fully available. The response payload provides the count of the resulting set. A new resultant set (view) which satisfies the input NAI restriction specification is created with the name <a href="#getViewName()"><code>viewName</code></a> passed in as part of the input.</p>
