A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#getRecordsFromCollection(com.gpudb.protocol.GetRecordsFromCollectionRequest)"><code>GPUdb.getRecordsFromCollection</code></a>. 
<p> Retrieves records from a collection. The operation can optionally return the record IDs which can be used in certain queries such as <a href="/content/api/java/com/gpudb/GPUdb#deleteRecords(com.gpudb.protocol.DeleteRecordsRequest)"><code>GPUdb.deleteRecords</code></a>. </p>
<p> This operation supports paging through the data via the <a href="#getOffset()"><code>offset</code></a> and <a href="#getLimit()"><code>limit</code></a> parameters. </p>
<p> Note that when using the Java API, it is not possible to retrieve records from join views using this operation. (DEPRECATED)</p>
