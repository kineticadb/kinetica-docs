<div className="block">A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#filterByTable(com.gpudb.protocol.FilterByTableRequest)"><code>GPUdb.filterByTable</code></a>. <p> Filters objects in one table based on objects in another table. The user must specify matching column types from the two tables (i.e. the target table from which objects will be filtered and the source table based on which the filter will be created); the column names need not be the same. If a <a href="#getViewName()"><code>viewName</code></a> is specified, then the filtered objects will then be put in a newly created view. The operation is synchronous, meaning that a response will not be returned until all objects are fully available in the result view. The return value contains the count (i.e. the size) of the resulting view.</p></div>
