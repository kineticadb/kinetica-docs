<div className="block">
A set of parameters for <a href="/content/api/java/com/gpudb/GPUdb#executeProc(com.gpudb.protocol.ExecuteProcRequest)"><code>GPUdb.executeProc</code></a>. 
<p> Executes a proc. This endpoint is asynchronous and does not wait for the proc to complete before returning. </p>
<p> If the proc being executed is distributed, <a href="#getInputTableNames()"><code>inputTableNames</code></a> and <a href="#getInputColumnNames()"><code>inputColumnNames</code></a> may be passed to the proc to use for reading data, and <a href="#getOutputTableNames()"><code>outputTableNames</code></a> may be passed to the proc to use for writing data. </p>
<p> If the proc being executed is non-distributed, these table parameters will be ignored.</p>
