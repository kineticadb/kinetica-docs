<div className="dl notes">
<div className="dt">Parameters:</div>
<div className="dd"><code>logLevel</code> - One of the supported log levels: TRACE, DEBUG, INFO, WARN, ERROR, OFF. <code>null</code> value is treated as 'OFF'.</div>
<div className="dt">Returns:</div>
<div className="dd"><code>true</code> if the log level was successfully set, <code>false</code> if no supported backend was detected or an error occurred.</div>
