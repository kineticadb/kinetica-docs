```
URL: http://<db.host>:<db.port>/create/directory
```
Creates a new directory in [KiFS](../../tools/kifs/). The new directory
serves as a location in which the user can upload files using
[/upload/files](/content/api/rest/upload_files_rest).

## Input Parameter Description

<ParamField body="directory_name" type="string">

Name of the directory in KiFS to be created.

</ParamField>

<ParamField body="options" type="map of string to strings">

Optional parameters.

The default value is an empty map ( \{\} ).

<Expandable title="options">

<ParamField body="create_home_directory">

When set, a home directory is created for the user name provided in the value. The input parameter *directory_name* must be an empty string in this case. The user must exist.

</ParamField>

<ParamField body="data_limit">

The maximum capacity, in bytes, to apply to the created directory. Set to -1 to indicate no upper limit. If empty, the system default limit is applied.

</ParamField>

<ParamField body="no_error_if_exists">

If *true*, does not return an error if the directory already exists.

The default value is `false`.

The supported values are:

- true
- false

</ParamField>

</Expandable>

</ParamField>
