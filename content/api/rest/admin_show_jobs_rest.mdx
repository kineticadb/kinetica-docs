```
URL: http://<db.host>:<db.port>/admin/show/jobs
```
Get a list of the current jobs in GPUdb.

## Input Parameter Description

<ParamField body="options" type="map of string to strings">

Optional parameters.

The default value is an empty map ( \{\} ).

<Expandable title="options">

<ParamField body="show_async_jobs">

If *true*, then the completed async jobs are also included in the response. By default, once the async jobs are completed they are no longer included in the jobs list.

The default value is `false`.

The supported values are:

- true
- false

</ParamField>

<ParamField body="show_worker_info">

If *true*, then information is also returned from worker ranks. By default only status from the head rank is returned.

The supported values are:

- true
- false

</ParamField>

</Expandable>

</ParamField>
