```
URL: http://<db.host>:<db.port>/create/role
```

Creates a new role.

**NOTE:** This endpoint should be used for on-premise deployments only.

## Input Parameter Description

<ParamField body="name" type="string">

Name of the role to be created. Must contain only lowercase letters, digits, and underscores, and cannot begin with a digit. Must not be the same name as an existing user or role.

</ParamField>

<ParamField body="options" type="map of string to strings">

Optional parameters.

The default value is an empty map ( \{\} ).

<Expandable title="options">

<ParamField body="resource_group">

Name of an existing resource group to associate with this user.

</ParamField>

</Expandable>

</ParamField>
