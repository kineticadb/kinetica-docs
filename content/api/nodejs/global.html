GPUdbCallback(err, response)

Callback function used for asynchronous GPUdb calls.
Parameters:
Name Type Description
err Error Object containing error information returned from the call. Will be null if no error occurred.
response Object Object containing any data returned from the call. Will be null if an error occurred.
